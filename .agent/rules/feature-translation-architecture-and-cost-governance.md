# Feature Rules: Translation Architecture and Cost Governance

## Normative priority
1) `sdd/core/constitution-canonical.md`
2) `.agent/rules/workspace-governance.md`
3) `sdd/features/translation-architecture-and-cost-governance/translation-architecture-and-cost-governance-spec-v1.md`

## Immutable rules
- Unificar traduccion de subtitulos, chat y TTS bajo backend gestionado.
- Toda degradacion por idioma no soportado debe ser explicita y trazable.
- Controlar coste por sesion y por feature con limites operativos.
- Mantener contratos tipados entre frontend y backend de traduccion.
